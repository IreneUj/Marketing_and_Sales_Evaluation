# Marketing_and_Sales_Evaluation
Data analysis and multiple linear regression project exploring the relationship between advertising spend and sales using Python.
# Advertising Spend and Sales Analysis
# Project Overview

This project analyzes the relationship between advertising expenditure and product sales using Python. The objective is to understand how spending on different advertising channels influences sales and to build a predictive model that estimates sales based on advertising investments.The project includes data cleaning, exploratory data analysis (EDA), visualization, correlation analysis, regression modeling.

# Objectives

Clean and prepare the advertising dataset for analysis.
Perform exploratory data analysis to identify trends and patterns.
Examine the relationship between advertising channels and sales.
Identify the advertising channel with the strongest impact on sales.
Build a multipe linear regression model to predict sales.
Evaluate the model using appropriate performance metrics.

# Project Workflow

Imported the dataset.
Cleaned missing and duplicate values.
Performed exploratory data analysis.
Created visualizations to understand the data.
Analyzed feature correlations.
Built a linear regression model.
Evaluated model performance

# Key Insights

TV advertising showed the strongest relationship with sales.
Radio advertising demonstrated a moderate relationship with sales.
Social Media advertising had a weaker relationship compared to TV and Radio.
The regression model was able to predict sales with good accuracy
